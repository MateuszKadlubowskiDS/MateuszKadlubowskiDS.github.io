Project 2 – Customer Churn Prediction (Telecom)

Goal: Predict whether a telecom customer will churn based on demographic, service, and billing data.

What I did: Built a full machine learning pipeline including data cleaning, feature engineering, exploratory data analysis, and model training using Logistic Regression, Regularized Logistic Regression (GLMNET), and Random Forest.
